# Support Matrix

## MVP (Required)
- Input container: zip
- Discoverable payloads: `.hhh`
- `.hhh` interpretation: Unity AssetBundle payloads
- Target ecosystem baseline: MoreHead-style cosmetics
- Unity baseline: 2022.3
- Output: in-browser GLB preview via three.js
- Hosting target: GitHub Pages (static hosting)

## Post-MVP (Planned)
- Broader Unity 6+ variants
- Expanded metadata extraction quality
- More robust fallback parse paths

## Never Supported
- Server-side parse pipeline
- Any backend API or server runtime dependency
- Cloud-hosted user libraries

## Out of Scope (MVP)
- Non-browser rendering targets
