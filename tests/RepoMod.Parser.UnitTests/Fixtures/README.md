# Parser Fixture Corpus

Place representative `.hhh` and `unitypackage` files used for parser compatibility verification.

## Goals
- Validate MoreHead-style Unity 2022.3 baseline.
- Catch regressions in bundle discovery and metadata extraction.
- Support fallback evaluation between parser sources.

## Data Handling
- Use fixtures you are legally allowed to test.
- Do not upload fixture mod content to project-controlled servers.
